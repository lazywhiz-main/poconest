Authentication

APIアクセスにはJWTトークンが必要です。以下の手順で認証を行ってください：

  • ユーザー登録またはログイン
  • アクセストークンの取得
  • リクエストヘッダーにトークンを設定
    • Bearer認証を使用
    • トークンの有効期限は24時間
💡 Info: セキュリティのため、HTTPSでの通信を強く推奨します。

トークン取得の例

// ログイン処理
const response = await fetch('/api/auth/login', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    username: 'developer@example.com',
    password: 'secure_password_123'
  })
});

const data = await response.json();
const token = data.access_token;

Rate Limiting

APIコールは1時間あたり1000回まで制限されています。制限を超えた場合は429 Too Many Requestsが返されます。

Code Examples

Python

# Python での API 呼び出し例
import requests

headers = {
    'Authorization': f'Bearer {token}',
    'Content-Type': 'application/json'
}

response = requests.get(
    'https://api.example.com/v2/users',
    headers=headers
)

if response.status_code == 200:
    users = response.json()
    print(f'取得したユーザー数: {len(users)}')

curl コマンド

# ユーザー一覧を取得
curl -X GET \
  https://api.example.com/v2/users \
  -H 'Authorization: Bearer YOUR_TOKEN_HERE' \
  -H 'Content-Type: application/json'
